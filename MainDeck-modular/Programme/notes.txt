1) Jedes Mal, wenn du im Terminal (für api) mit dem Backend arbeitest, musst du das venv aktivieren:
  cd ~/NoteDeck/MainDeck-modular/api
  source venv/bin/activate
Dann steht im Terminal

  a)API:
  cd ~/NoteDeck/MainDeck-modular/api
  source venv/bin/activate
  uvicorn main:app --reload
  b)React
  cd ~/NoteDeck/playground
  npm run dev
